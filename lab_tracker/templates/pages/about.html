Lab Tracker

Electronics Engineering Student Society has successfully conducted phase II of ideathon-an idea-collection and project development program in the previous week, where we had an idea pitching session through which we collected innovative ideas. Based on the problem analysis and feasibility study, the participants are proceeding towards their project development phase with the project "Lab Tracker". 'Lab Tracker' is a web application that allows students to view lab resources online, learn specifications of resources, trace availability and many features for everyone. This project is being done by a team of BEI first-year students having team members Ritik Krishna Shrestha, Yukesh Nepal, Bikesh Maharjan and Prashant Soni.
